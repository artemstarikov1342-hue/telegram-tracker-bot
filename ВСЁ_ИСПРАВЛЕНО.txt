ЧТО ИСПРАВЛЕНО:
─────────────────────────────────────────────────────────────

✅ Задачи партнеров теперь СОХРАНЯЮТ тег (WEB2, WEB3, etc)
✅ /partners показывает всех партнеров с задачами
✅ /partner WEB2 показывает задачи конкретного партнера
✅ Улучшена фильтрация задач


╔═══════════════════════════════════════════════════════════════╗
║              ШАГ 1: ОЧИСТИ СТАРЫЕ ЗАДАЧИ                      ║
╚═══════════════════════════════════════════════════════════════╝

Дважды кликни:

   ОЧИСТИТЬ_БД.cmd

Это удалит старые задачи из БД (MNG-2, MNG-3, MNG-4, MNG-5)

Ты увидишь:

   ❌ Удаляю: MNG-2 (личка, MNG, open)
   ❌ Удаляю: MNG-3 (личка, MNG, open)
   ...
   ✅ Удалено задач: 4
   ✅ БД обновлена!


╔═══════════════════════════════════════════════════════════════╗
║              ШАГ 2: ПЕРЕЗАПУСТИ БОТА                          ║
╚═══════════════════════════════════════════════════════════════╝

1. Закрой окно с ботом
2. Дважды кликни: START_BOT.cmd
3. Дождись "Бот запущен..."


╔═══════════════════════════════════════════════════════════════╗
║              ШАГ 3: СОЗДАЙ НОВЫЕ ТЕСТОВЫЕ ЗАДАЧИ              ║
╚═══════════════════════════════════════════════════════════════╝

В группе напиши:

   #задача WEB#2 тест партнера 2
   
   #задача WEB#3 тест партнера 3
   
   #задача WEB#5 тест партнера 5


╔═══════════════════════════════════════════════════════════════╗
║              ШАГ 4: ПРОВЕРЬ КОМАНДЫ                           ║
╚═══════════════════════════════════════════════════════════════╝

Напиши боту:

   /partners

ДОЛЖНО ПОКАЗАТЬ:

   🔹 WEB5: 1 задач(и)
   
   💡 Всего партнеров: 3
   📋 Всего задач: 3
   
   Используйте /partner WEB2 для деталей


Напиши боту:

   /partner WEB2

ДОЛЖНО ПОКАЗАТЬ:

   📋 Задачи партнера WEB2 (1):

║              ЕСЛИ ЧТО-ТО НЕ РАБОТАЕТ                          ║
╚═══════════════════════════════════════════════════════════════╝

/partners не показывает задачи:
   → Убедись что ШАГ 1 выполнен (очистка БД)
   → Создай новые задачи (ШАГ 3)
   → Перезапусти бота

/partner WEB2 не показывает:
   → Создай задачу с WEB#2 ПОСЛЕ перезапуска бота
   → Проверь логи бота (должно быть: "✅ Найден ID партнера: 2")

/mytasks показывает старые задачи:
   → Запусти ОЧИСТИТЬ_БД.cmd ещё раз
   → Перезапусти бота
